# Bootstraping

- Init typescript project

```jsx
npm init -y
npx tsc --init
```

- Change rootDir and outDir

```jsx
"rootDir": "./src",
"outDir": "./dist"
```

- Install dependencies for express

```jsx
npm i express 
```
